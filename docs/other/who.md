# 我是谁

```bash
who am i            // 查看当前登录用户
who mom likes       // 查看当前登录用户
whoami              // 查看当前登录用户
```
```bash
who -b              // 查看系统最后一次启动时间
who -d              // 查看死掉的进程
who -l              // 查看登录进程
who -m              // 查看当前登录用户
who -q              // 统计登录用户
who -r              // 查看当前运行级别
who -T              // 显示终端状态
who -u              // 显示用户登录状态
```
