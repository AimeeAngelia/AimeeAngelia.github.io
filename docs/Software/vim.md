# Vim

## Vim的三种工作模式

1. 命令模式(Command mode)
	所敲按键，编辑器均会理解为命令，以命令驱动执行不同功能。
2. 输入模式(Insert mode)
	又名编辑模式、插入模式。
	可以对文件内容进行自由编辑。
3. 底线命令模式(Last line mode)
	以`:`为起始，通常用于文件的保存，退出。
